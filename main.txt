Git Command Instructions

1 - Save file
2 - Add changed files to local repository ->   git add .
3 - Commit what you did ->                     git commit -m "MESSAGE DESCRIPTION"
4 - Send to the remote repository ->           git push origin main
5 - Request changes update local repository -> git pull origin main
